# Quick Start

Create your first MCP server in under 5 minutes.

<Steps>
<Step>

### Install

```bash
npm install mcpcraft-sdk zod
```

</Step>

<Step>

### Write your server

Create `server.ts`:

```typescript
import { createServer, tool } from "mcpcraft-sdk"

const server = createServer({ name: "my-first-server" })

server.add(tool({
  name: "greet",
  description: "Greets a user by name",
  input: {
    name: { type: "string", description: "The user's name" }
  },
  run: async ({ name }) => {
    return { message: `Hello, ${name}!` }
  }
}))

server.start()
```

</Step>

<Step>

### Run it

```bash
npx ts-node server.ts
```

Your server is now listening on stdio.

<Callout type="info">
  Because the server communicates via standard input/output, running it in the terminal will start a blocking process. To test it, connect it to an MCP host like Claude Desktop.
</Callout>

</Step>
</Steps>

## What just happened?

- `createServer({ name: "my-first-server" })` — initialized an MCP server
- `tool({ ... })` — defined a callable function the LLM can invoke
- `input: { name: { type: "string", ... } }` — declared a parameter (mcpcraft-sdk builds the JSON Schema + Zod validator automatically)
- `run: async ({ name }) => ...` — the handler that executes when the LLM calls the tool
- `server.start()` — launched the server on stdio transport

<Callout type="success">
  That's everything you need. Any MCP-compatible client — Claude Desktop, VS Code Copilot, Cursor — can now discover and call your `greet` tool.
</Callout>
