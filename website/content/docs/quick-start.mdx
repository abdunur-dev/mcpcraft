# Quick Start

Create your first Model Context Protocol (MCP) server in under 5 minutes.

<Steps>
<Step>

### Install Dependency

Install `mcpcraft` and `zod` in your project folder:

```bash
npm install mcpcraft-sdk zod
```

</Step>

<Step>

### Create Your Server

Create a file named `server.ts` and define your first tool:

```typescript
import { createServer, tool } from "mcpcraft-sdk";

// 1. Initialize the server
const server = createServer({
  name: "my-first-server",
  version: "1.0.0"
});

// 2. Register a tool with type-safe schema validation
server.add(
  tool({
    name: "greet",
    description: "Greets a user by name",
    input: {
      name: { type: "string", description: "The name of the user to greet" }
    },
    run: async ({ name }) => {
      return { message: `Hello, ${name}!` };
    }
  })
);

// 3. Start the server on stdio transport
server.start();
```

</Step>

<Step>

### Run the Server

You can execute your server using `ts-node` or compile it with `tsc`:

```bash
npx ts-node server.ts
```

Your server is now listening on standard input/output (stdio) streams.

<Callout type="info">
  Because the server communicates via standard input/output, running it in the terminal will start a blocking process. To test it, connect it to an MCP host like Claude Desktop.
</Callout>

</Step>
</Steps>
