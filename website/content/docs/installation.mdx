# Installation

Install `mcpkit` via npm:

```bash
npm install mcpkit zod
```

## System Requirements

- **Node.js**: version 18.0.0 or higher
- **TypeScript**: version 5.0.0 or higher

## Under the Hood

`mcpkit` is built on top of the official `@modelcontextprotocol/sdk`. When you install `mcpkit`, the official SDK is installed automatically as a dependency. 

You do not need to write any low-level JSON-RPC boilerplate or manually manage Stdio transport connections; `mcpkit` handles it all for you.
