# Installation

Install `mcpcraft` via npm:

```bash
npm install mcpcraft zod
```

## System Requirements

- **Node.js**: version 18.0.0 or higher
- **TypeScript**: version 5.0.0 or higher

## Under the Hood

`mcpcraft` is built on top of the official `@modelcontextprotocol/sdk`. When you install `mcpcraft`, the official SDK is installed automatically as a dependency. 

You do not need to write any low-level JSON-RPC boilerplate or manually manage Stdio transport connections; `mcpcraft` handles it all for you.
