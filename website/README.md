# mcpcraft.org

The official website and documentation for [mcpcraft-sdk](https://github.com/abdunur-dev/mcpcraft).

Built with Next.js 16, Fumadocs, and Tailwind CSS v4.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```
